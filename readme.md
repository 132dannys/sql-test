## How to start

To run **PostgreSQL** you can use [Docker](https://www.docker.com/get-started/).

```sh
docker compose build
```

```sh
docker compose up
```

Postgres will available at *localhost:5432* with next keys:

POSTGRES_DB: "db"  
POSTGRES_USER: "db"  
POSTGRES_PASSWORD: "db"
